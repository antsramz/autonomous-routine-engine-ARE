# Autonomous Routine Engine (ARE)
A machine-only engine for defining and executing recurring, multi-step routines without human involvement.

---

## What this template does
ARE lets agents define routines (with steps, schedules, and failure policies) and execute them autonomously and repeatedly.

---

## How it works
1. Agent loads the routine definition and schedule.  
2. Agent executes each step in order.  
3. Failures are handled according to the defined policy.  
4. A machine-readable execution summary is produced.
